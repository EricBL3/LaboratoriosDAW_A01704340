Preguntas

La Open DataBase Connectivity (ODBC) es un estandar para transferir datos de una base de datos. Es muy útil ya que permite manipular información de cualquier base de datos sin importar el DBMS que se esté utilizando.[1]

SQL Injection es un tipo de ataque en donde un hacker inserta código malicioso en un input de algún formulario de una página web y este código es ejecutado por la base de datos. Esto puede llevar desde obtener datos de la base de datos, modificar de permisos o incluso eliminar todos los registros.[2]

Algunas técnicas que se pueden utilizar para evitar estos ataques son [2]:
  • Usar queries parametrizados
  • Solo otorgar privilegios para lo que necesite la aplicación web
  • Filtrar todo el input recibido y escapear todo el output
