Ajax es muy importante para el desarrollo de las RIA's ya que es la tecnología que permite que la interactividad sea extremadamente rápida y que parezca una aplicación de escritorio. Esto se logra gracias a que Ajax permite hacer llamadas asíncronas a la base de datos para solo recuperar y recargar una parte específica de la página web.

Las valoraciones de seguridad tienen que seguir realizandose del lado del servidor por medio de controladores php con htmlspecialchars() y del lado del cliente con validaciones de js o con regex de los inputs para asegurarse que lo que manda el usuario es correcto.

JSON o JavaScript Object Notation por sus siglas en inglés, es un formato de texto ligero de intercambio de información. Es fácil de leer y escribir para humanos y fácil de interpretar y generar para computadoras por lo que es un formato muy utilizado para intercambiar información.[1]
Está compuesto de dos estructuras:
  • Una colección de pares de nombres y valores
  • Una lista ordenada de valores
