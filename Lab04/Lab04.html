Preguntas

  1. ¿Qué diferencias y semejanzas hay entre Java y JavaScript?[1]

    Diferencias:

    • Java debe ser compilado mientras que JavaScript es interpretado.
    • JavaScript solo puede correr en un navegador.
    • Java es un lenguaje de programación mientras que JavaScript es un lenguaje de scripting.

    Semejanzas:

    • Ambos son orientados a objetos.
    • Ambos tienen soporte multiplataforma.
    • Java también puede correr en un navegador.

  2. ¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)[2]

    • toTimeString(): Regresa la hora y el horario actual.
    • toDateString(): Regresa la fecha actual.
    • getFullYear(): Regresa el año actual.
    • getMonth(): Regresa el mes actual.
    • toUTCString(): Convierte el objeto Date a un string con formato UTC

  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)[2]

    • reduce(): Aplica una función a un acumulador y cada valor del arreglo para reducirlo a un solo valor.
    • filter(): El método recibe una función con una condición como parámetro y regresa los valores del arreglo que pasaron la condición.
    • reverse(): Invierte un arreglo.
    • concat(): Concatena el arreglo pasado como parámetro.
    • shift(): Remueve el primer elemento de un arreglo.

  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    Se declara con let.

  5. ¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

    Que esa variable no va a desaparecer en cuanto acabe el bloque por lo que puede resultar problemático si se usa una variable con el mismo nombre después.

  6. ¿Qué método de String se puede utilizar para buscar patrones con expresiones regulares?¿Para qué podrías utilizar esto en una aplicación web?[3]

    Los métodos match() y search(). Estos métodos podrían ser útiles para realizar búsquedas de texto en la aplicación web.
