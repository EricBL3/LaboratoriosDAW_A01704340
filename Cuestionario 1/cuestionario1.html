Preguntas

  1. ¿Qué es entonces una base de datos - DB?

    Una base de datos es una colección de archivos interrelacionados en donde aparecen multiples ocurrencias de un mismo tipo de registro. [1]

  2. ¿En qué casos es conveniente usar bases de datos?

    Es conveniente usar una base de datos cuando varios usuarios usan archivos interrelacionados y cuando el volumen de los datos es un factor importante.[1]
    Otros casos en los que es conveniente usar una base de datos son: [2]
    • Cuando los datos tienen que mantenerse exactos y consistentes.
    • Cuando se hacen cambios que van a afectar a varias entidades
    • Cuando se necesita un sistema escalable

  3. ¿Qué es un sistema de gestión de base de datos - DBMS?

    Es una colección de datos interrelacionados y un conjunto de programas para acceder a los datos. Además, proporciona un entorno conveniente y eficiente para los usuarios.

  4. Enlista y explica las funciones/responsabilidades que tienen los DBMS.

    • Implantación de la integridad: Al especificar restricciones, los DBMS pueden determinar si las actualizaciones a la base de datos violan una restricción para tomar la medida adecuada.
    • Implantación de la seguridad: No todos los usuarios deben ser capaces de accesar a todo el contenido de la base de datos.
    • Copia de seguridad y recuperación: Se realizan estas copias para prevenir la pérdida total de información en caso de fallos o errores.
    • Control de concurrencia: Controlar la interacción entre usuarios concurrentes para prevenir errores de inconsistencia.

  5. Ejemplifica y justifica en que proyecto de los que has realizado hubiera sido conveniente utilizar una base de datos.
