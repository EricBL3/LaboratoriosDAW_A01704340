Preguntas

Ventajas[1]:
  • Los servicios web hacen más fácil la creación de aplicaciones web ya que puedes usar herramientas creadas por otras personas/empresas en vez de reinventar todo.
  • Los servicios web permiten que varias aplicaciones se comuniquen entre si por lo que hacen que la aplicación web sea independiente de la tecnología usada.
  • Todos los servicios web usan un protocolo estandarizado para comunicarse.
Desventajas[2]:
  • No es seguro que un servicio web esté disponible las 24 horas del día por lo que las funciones que usen servicios web en una aplicación pueden llegar a fallar.
  • Los servicios web están hechos para dar soluciones generales por lo que es muy difícil que exista alguno que de una funcionalidad muy específica en una aplicación web.
  • Las funciones que usan servicios web se vuelven completamente dependientes de los desarrolladores de este servicio ya que pueden llegar a tener errores si hay algún cambio en la manera en la que funciona el servicio web.
